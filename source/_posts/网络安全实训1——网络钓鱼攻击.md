## 利用 nodejs 开启静态服务

> 补充利用 nodejs 开启静态服务
>
> ```bash
> sudo apt update
> sudo apt install nodejs npm
> ```
>
> 安装完成后进入你的静态文件所在目录
>
> 开启命令行
>
> ```bash
> http-server
> ```
